Overhead Vision Guided Quadruped Project

The basic idea of this project is that an overhead camera system takes a stream of images of the quadruped. The quadruped's center is located in the image from the vision target attached to it. A user defined shape is then put over the frame. Using a Kalman filter and PID the quadruped will be kept inside the lines of the user defined shape. A small desktop will compute the necessary trajectory changes and send the according serial commands over an RF transceiver to the robot. More to come soon! I plan to make this my CSH major project for the 2016-2017 year.
